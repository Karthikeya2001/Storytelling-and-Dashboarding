# Storytelling with Visuals

A collection of visual storytelling techniques created in Excel to demonstrate how charts and data visualization can improve analytical communication, highlight business impact, and support decision-making.

---

## Overview

This project showcases different storytelling approaches using charts and structured datasets. The workbook focuses on presenting insights clearly instead of only displaying raw numbers.

The examples are designed around realistic business and analytics scenarios such as:

- Impact analysis
- Forecasting trends
- Comparative performance analysis
- Highlighting changes after interventions
- Stakeholder-focused storytelling

---

## Project Objectives

The main objective of this project is to demonstrate how visual storytelling can:

- Improve clarity of insights
- Direct stakeholder attention toward important events
- Make trends easier to understand
- Support data-driven decisions
- Simplify analytical communication

---

## Storytelling Principles Covered

The workbook highlights important storytelling fundamentals:

1. Understanding the intent behind the analysis
2. Understanding the product and data context
3. Understanding stakeholder expectations
4. Designing visuals that guide attention effectively
5. Presenting insights with clarity and purpose

---

### 2. Shaded Areas

Demonstrates how shaded areas can be used in charts to highlight specific periods or interventions.

#### Scenario

Comparing newsletter CTR between a control group and pilot group after an enhancement rollout.

#### Purpose

To visually draw attention toward the time period where the enhancement was introduced.

#### Concepts Demonstrated

- Highlighting critical business periods
- Before vs after comparison
- Attention guidance using visual emphasis
- Contextual storytelling

---

### 3. Clustered Comparison

Demonstrates comparative analysis using clustered charts.

#### Scenario

Comparing unresolved ticket counts across multiple teams after process optimization.

#### Purpose

To clearly communicate that Team A reduced unresolved tickets more effectively compared to other teams.

#### Concepts Demonstrated

- Comparative storytelling
- Performance benchmarking
- Visual prioritization
- Operational impact communication

---

### 4. Forecasting

Demonstrates forecasting and trend storytelling.

#### Scenario

Predicting future product sales trends using historical data.

#### Purpose

To communicate expected upward trends in product demand.

#### Concepts Demonstrated

- Forecast visualization
- Trend communication
- Business projection storytelling
- Analytical decision support
